Jenkins + Kubernetes: Fast CI at Scale

Key practices for reliable, scalable CI/CD on Kubernetes with Jenkins agents.

  • Use ephemeral agents with node selectors and resource requests/limits.
  • Cache dependencies via shared volumes or repo-level caching.
  • Externalize secrets (e.g., K8s Secrets, Vault) and avoid inline credentials.
  • Parallelize test stages and use post actions for consistent cleanup.
